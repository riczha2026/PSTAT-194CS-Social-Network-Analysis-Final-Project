Random Walk Sampling Code

set.seed(123)

subgraphs <- list()

for(i in 1:4){
  start_node <- sample(V(graph), 1)
  walk <- random_walk(graph,
                      start = start_node,
                      steps = 500,
                      mode = "all")
  nodes_sampled <- unique(walk)
  nodes_sampled <- nodes_sampled[1:200]
  subgraphs[[i]] <- induced_subgraph(graph, nodes_sampled)
}

Community Detection

  • Walktrap detects more communities than other algorithms
    • Fast greedy and Louvain typically struggle to see smaller communities
    • Implies a hierarchical/subcommunity structure
  • Stable results with modularity of around 0.7 or higher
  • Largest community is less than a quarter of entire network size

Adjacency Matrix

  • 200x200 matrix where each cell = 1 if a hyperlink exists between two pages
